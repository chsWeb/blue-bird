## Restore Blue Bird with Amino
Using the Amino extension for Chome to restore the Twitter Blue Bird logo
1. Install the Amino extension (BTW, it is incredible)
2. Laucnh Web Inspector
3. Inspect the Twitter logo
4. Choose Amino in
